# âœ… Todo List Application

This is a simple web-based Todo List built using HTML, CSS, and JavaScript.  
Users can:
- Add new tasks
- Edit specific tasks
- Delete any task
- Check off completed tasks
- Save all tasks using localStorage so they don't disappear on reload
